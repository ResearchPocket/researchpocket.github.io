Product overview

A private place for the useful parts of the web.

ResearchPocket saves URLs together with the title, note, tags, and favorite state you choose. The library works offline and remains on your device unless you explicitly connect private sync.

Local-first by design

Your devices build the library.

IndexedDB and SQLite remain local materialized views. Private sync transports immutable CRDT updates, so application state converges without Git deciding which edit wins.

There is no ResearchPocket account, hosted application database, advertising profile, analytics script, or AI-generated memory feed.
