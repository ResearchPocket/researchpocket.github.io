Useful links deserve a place you control.

ResearchPocket is a human-first library for saving URLs with the context that made them matter. It works locally, stays private by default, and needs no ResearchPocket account or always-on backend.

Built for remembering

Your library is not an AI memory feed.

ResearchPocket does not autonomously decide what you know. You choose the links, titles, notes, tags, favorites, and deletions. Those human decisions remain canonical across every interface.

The product stores URLs and your authored context—not a copy of every page, not an advertising profile, and not a hosted knowledge graph.

No application server

The remote stores updates. Your devices build the library.

SQLite and IndexedDB stay local. A private repository holds immutable, uniquely addressed CRDT updates so offline devices can meet again without Git commits choosing a winner.
